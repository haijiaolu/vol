# 《法华经》（妙法莲华经）

### 《妙法莲华经·观世音菩萨普门品第二十五》

::: info

> [!quote]
>
> 福不唐捐
>
> 衍生：
> - 日拱一卒，功不唐捐。
> - 日拱一卒无有尽，功不唐捐终入海。

:::

```
若有众生，恭敬礼拜观世音菩萨，福不唐捐。是故众生，皆应受持观世音菩萨名号。
```

# 《后汉记》

### 《后汉记·卷二十三孝灵皇帝纪上》

:::info

> [!quote]
>
> 经师易遇，人师难遭。——魏昭

:::

```markdown
袁宏《后汉记·卷二十三孝灵皇帝纪上》
尝止陈国文孝，童子魏昭求入其房供给洒扫，泰曰：“年少当精义书，曷为（来）〔求〕近我乎？”
昭曰：“盖闻经师易遇，人师难遭，故欲以素丝之质附近朱蓝耳。”泰美其言，听与共止。

> 备注：
> * 昭：魏昭；
> * 泰：陈泰。
```

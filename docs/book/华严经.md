# 《华严经》

### 《大方广佛华严经卷》（华严经）

- 《大方广佛华严经卷第十九·升夜摩天宫品第十九》：“如菩萨初心，不与后心俱，智无智亦然，二心不同时。”
- 《大方广佛华严经卷第三十七·十地品第二十六之四》：“无明，所作业果是行，行依止初心是识，与识共生四取蕴为名色，名色增长为六处，……”

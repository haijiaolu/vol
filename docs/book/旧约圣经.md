# 《旧约圣经》

### 名言：太阳底下没有新鲜事。

:::info

> [!quote]
>
> There is nothing new under the sun.
> > 太阳底下没有新鲜事。

:::

* 汉语翻译：
```
旧约（和合本修订本）
传道书1
人生的虚空

9
已有的事，后必再有；
已行的事，后必再行。
日光之下并无新事。
```

* 英语翻译：
```
The Bible (KJV)
Ecclesiastes 1:9

The thing that hath been, it is that which shall be;
and that which is done is that which shall be done:
and there is no new thing under the sun.
```

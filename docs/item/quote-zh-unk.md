# 未知的格言摘录（中文）

## 出处不明 1

::: warning

> [!warning]
>
> 勤学如春起之苗，不见其增，日有所长。辍学如磨刀之石，不见其损，日有所亏。——陶渊明。

:::

或言出自《陶渊明集》。找了几种电子版资料，均未找到。
《陶渊明集》大部分是诗歌，文赋篇章不多，句式风格也有点奇怪，大概是伪托之作。

# 冯唐 （张海鹏）

### 名句

::: info

> [!quote]
>
> 春水初生，春林初盛。春风十里，不如你。——冯唐

:::

```
《三十六大》
24 大喜

其三十，大酒喝到身体摇晃，勉强不坠地，一时，脑壳里杂乱沸腾如重庆火锅，似乎见一奸人在面前，
无遮拦狂骂，骂到奸人消失，又狂发短信和微博，又抓笔抓纸写诗。次日酒醒，头痛如上紧箍咒。
电话给那奸人，侧面了解，发现奸人昨晚不在，全是幻觉。查短信和微博记录，完全没有，
昨晚手机早已没电。查床头，纸笔还在，字迹尚可辨认：“春水初生，春林初盛。春风十里，不如你”，
诗句大好。于是欢喜。

---

《冯唐诗百首》

春

春水初生
春林初盛
春风十里，不如你
```

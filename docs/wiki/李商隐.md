# 李商隐

### 《暮秋独游曲江》

```
《暮秋独游曲江》
〔唐代〕 李商隐

荷叶生时春恨生，荷叶枯时秋恨成。
深知身在情长在，怅望江头江水声。
```

### 《晚晴》

```
《晚晴》
〔唐代〕李商隐

深居俯夹城，春去夏犹清。
天意怜幽草，人间重晚晴。
并添高阁迥，微注小窗明。
越鸟巢干后，归飞体更轻。
```

# 赖内·马利亚·里尔克（Rainer Maria Rilke）

Rainer Maria Rilke （1875-1926）奥地利（奥匈帝国）诗人。

### 《挽歌——为沃尔夫伯爵封·卡尔克洛伊特而作》

::: info

> [!quote]
>
> Wer spricht von Siegen? Überstehn ist alles.
>
> > 有何胜利可言？挺住就是着一切。——里尔克  
> > （谁能轻言胜利？忍耐就是一切。）  
> > （有谁在谈胜利呢？忍耐就是一切。）  
> > （谁还在言及胜利呢？忍耐就是一切。）  

:::

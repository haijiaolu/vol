# 邱琼

### 《这一世木已成舟》：如果有来生

```
如果有来生，要做一棵树，
站成永恒。没有悲欢的姿势，
一半在尘土里安详，
一半在风里飞扬；
一半洒落荫凉，
一半沐浴阳光。
非常沉默、非常骄傲。
从不依靠、从不寻找。

> 来自三毛的那首诗《如果有来生》：https://book.douban.com/review/7842425/
```

《这一世木已成舟》

第二十一章 文/邱琼
平安夜到了，漓江给琥珀送了礼物。晓珀撕开墨绿褶皱纸包装的礼盒，里面是圣诞树形状的香水
瓶，三宅一生的牌子。

“这是二00一年的纪念版”，漓江微笑着说，“名字很好听，叫做’一生之树”。”他还记得琥珀曾经说
过，如果有来生，要做一棵树，站成永恒，没有悲欢的姿势。一半在生土里安详，一半在风里飞扬，
半洒落阴凉，一半沐浴阳光。非常沉默非常骄傲，从不依靠从不寻找，是她的榜样。

这跟阿燃每年送给她的礼物是多么不一样。从初识起，阿燃就带给她又香又白的花朵。通常是白色
的维菊，缀满天真无邪的白色细长花瓣。琥珀从杂志上了解到维药的花语是“简单的一天”。那好象属于
某个青涩无端的年纪。

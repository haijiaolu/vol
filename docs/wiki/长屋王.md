# 長屋王（长屋王）

長屋王，ながやおう/ ながやのおおきみ（684—729），日本奈良时代皇族。

### 《绣袈裟衣缘》

```
山川异域，风月同天。寄诸佛子，共结来缘。
——【日本】长屋王《绣袈裟衣缘》，（全唐诗：卷732-11）

> 原题注：明皇时，长屋尝造千袈裟，绣偈于衣缘，来施中华。真公因泛海至彼国传法焉。
> 出自《唐大和上东征传》
> 收录在日本现存最早的汉诗集《怀风藻》
```

# 张超（黑眼睛）

### 《村中油菜花开》

```
《村中油菜花开》
张超（黑眼睛）

绵绵十里绕孤村，坐爱馨香不闭门。
春日闲居无一事，独看花海到黄昏。
```

### 《庚子立春》

```
《庚子立春》
张超（黑眼睛）

独守晴窗困一隅，避瘟终日少欢娱。
东君不晓人间事，又遣庭前草木苏。
```

### 《离家五百里》

```
《离家五百里》
张超（黑眼睛）

行役经年岁，游子今始归。
去时花灼灼，来时雪霏霏。
故园日以近，中心日以悲。
吉他久已弊，缁尘染素衣。
离家五百里，铁轨何逶迤。
离家四百里，北风漠漠吹。
离家三百里，旷野行人稀。
离家两百里，穷巷在山陲。
离家一百里，慈母应倚扉。
```

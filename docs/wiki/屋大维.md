# Gaius Octavius Augustus（盖维斯·屋大维·奥古斯都）

Augustus（公元前 63 年—公元 14），罗马帝国创始人。

### 名言

:::info

> [!quote]
>
> Festina lente.
>
> > 慢慢地，快进。——奥古斯都（屋大维）座右铭  
> > 【其他翻译】慢慢地赶。

:::

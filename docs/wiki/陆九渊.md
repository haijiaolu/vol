# 陆九渊

### 《六经》注我，我注《六经》

:::info

> [!quote]
>
> 《六经》注我，我注《六经》。

:::

```
《陆九渊集·卷三十四·语录上·傅子云季鲁录》

或问﹕“先生何不著书？”对曰﹕“《六经》注我，我注《六经》。”
韩退之是倒做，盖欲因学文而学道。欧公极似韩，其聪明皆过人，然不合初头俗了。
或问﹕“如何俗了？”曰﹕“符读书城南三上宰相书是已。至二程方不俗，然聪明却有所不及。”
```

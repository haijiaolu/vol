# 邵雍

### 《观盛花吟》

```markdown
《观盛花吟》
〔宋代〕 邵雍

纷纷五代乱离间，一旦云开复见天。
草木百年新雨露，车书万里旧江山。
寻常巷陌陈罗绮，几处楼台奏管弦。
天下太平无事日，鸯花无限日高眠。

> 鸯花无限日高眠。-> 莺花无限日高眠。

> 纷纷五代乱离间，一旦云开复见天。
> 草木百年新雨露，车书万里旧江山。
> 寻常巷陌陈罗绮，几处楼台奏管弦。
> 人乐太平无事日，莺花无限日高眠。
> ——《水浒传》引首

```

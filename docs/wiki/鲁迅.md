# 鲁迅

### 《题三义塔》

```markdown
《题三义塔》
鲁迅

> 三义塔者，中国上海闸北三义里遗鸠埋骨之塔也，在日本，农人共建。
>
> 西村博士于上海战后得丧家之鸠，持归养之，初亦相安，而终化去。
> 建塔以藏，且征题咏，率成一律，聊答遐情云尔。

奔霆飞熛歼人子，败井颓垣剩饿鸠。
偶值大心离火宅，终遗高塔念瀛洲。
精禽梦觉仍衔石，斗士诚坚共抗流。
度尽劫波兄弟在，相逢一笑泯恩仇。
```

### 《热风》

```
无论什么黑暗来防范思潮，什么悲惨来袭击社会，什么罪恶来亵渎人道，
人类的渴仰完全的潜力，总是踏了这些铁蒺藜向前进。
——鲁迅《热风•随感录第六十六（生命的路）》
<http://www.luxunmuseum.com.cn/cx/content.php?id=1439>

所以我时常害怕，原中国青年都摆脱冷气，只是向上走，不必听自暴自弃者流的话。
能做事的做事，能发声的发声。有一分热，发一分光，就令萤火一般，也可以在黑暗里发一点光，
不必等候炬火。此后如竟没有炬火：我便是唯一的光。倘若有了炬火，出了太阳，
我们自然心悦诚服的消失，不但毫无不平，而且还要随喜赞美这炬火或太阳；
因为他照了人类，连我都在内。我又愿中国青年都只是向上走，不必理会这冷笑和暗箭。
—— 鲁迅《热风·随感录四十一》
（本篇最初发表于一九一九年一月十五日《新青年》第六卷第一号，署名唐俟。）
<http://www.luxunmuseum.com.cn/cx/content.php?id=1457>

做了人类想成仙；生在地上要上天；明明是现代人，吸着现在的空气，
却偏要勒派朽腐的名教，僵死的语言，侮蔑尽现在，这都是“现在的屠杀者”。
杀了“现在”，也便杀了“将来”。——将来是子孙的时代。
—— 鲁迅《热风·随感录五十七》
（杂文集《热风》系1925年编定，本篇《随感录57》
最初发表于一九一九年五月《新青年》第六卷第五号，署名唐俟。）
```

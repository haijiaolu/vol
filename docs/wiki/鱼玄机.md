# 鱼玄机（鱼幼薇）

### 《赠邻女》

```
《赠邻女》
〔唐代〕鱼玄机

羞日遮罗袖，愁春懒起妆。
易求无价宝，难得有心郎。
枕上潜垂泪，花间暗断肠。
自能窥宋玉，何必恨王昌？
```

---

### 《打球作》

```
《打球作》（打毬作）
〔唐代〕鱼玄机

坚圆净滑一星流，月杖争敲未拟休。
无滞碍时从拨弄，有遮栏处任钩留。
不辞宛转长随手，却恐相将不到头。
毕竟入门应始了，愿君争取最前筹。
```

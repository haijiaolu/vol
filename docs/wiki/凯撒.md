# Gaius Iulius Caesar（盖乌斯·尤利乌斯·恺撒）

盖乌斯·尤利乌斯·恺撒（拉丁语：Gaius Iulius Caesar；前 100 年—前 44 年），罗马共和国末期的军事统帅、政治家

### 格言

::: info

> [!quote]
>
> Veni, vidi, vici
>
> > I came, I saw, I conquered.
> > 我来，我看见，我征服。

:::

出处：
凯撒在前 47 年泽拉（今土耳其济莱附近）战役中打败本都（古代小亚细亚北部的一个地区，在黑海南岸）国王法尔纳克二世之后写给罗马元老院的著名捷报。

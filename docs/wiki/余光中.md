## 出处不明的诗句

### 《你的眼睛真好看》

```
你的眼睛可真好看
里面有晴雨、日月、山川、江河、云雾、花鸟
但我的眼睛更好看
因为我的眼里有你
—— 余光中《你的眼睛真好看》
```

### 《绝色》

```
若逢新雪初霁 满月当空
下面平铺着皓影
上面流转着亮银
而你带笑的向我走来
月色与雪色之间
你是第三种绝色
——《绝色》
```

### 《余光中写给妻子的信》

```
不要问我心里有没有你
我余光中都是你
——《余光中写给妻子的信》
```

### 《独白》

```
月光还是少年的月光
九州一色还是李白的光
——《独白》
```

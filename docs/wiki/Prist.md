# Prist（刘垚）


### 《生查子》

```markdown
《生查子》
于丘烽

平江柳色青，花月遥相守。
岁岁复年年，逢此冰消后。
几回沧海平，山雪别云岫。
一眼万年轻，唯此心如旧。

> Prist小说《天涯客·第六十章 夫妻、第六十九章 回归》  
> 于丘烽为小说中人物
```

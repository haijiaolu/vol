# 刘琨

### 《重赠卢谌》

```
《重赠卢谌》
〔魏晋〕 刘琨

握中有悬璧，本自荆山璆。
惟彼太公望，昔在渭滨叟。
邓生何感激，千里来相求。
白登幸曲逆，鸿门赖留侯。
重耳任五贤，小白相射钩。
苟能隆二伯，安问党与雠？
中夜抚枕叹，想与数子游。
吾衰久矣夫，何其不梦周？
谁云圣达节，知命故不忧。
宣尼悲获麟，西狩涕孔丘。
功业未及建，夕阳忽西流。
时哉不我与，去乎若云浮。
朱实陨劲风，繁英落素秋。
狭路倾华盖，骇驷摧双辀。
何意百炼刚，化为绕指柔。
```

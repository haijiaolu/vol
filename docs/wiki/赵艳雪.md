# 赵艳雪

### 《和查为仁悼亡诗》

```markdown
《和查为仁悼亡诗》
（又说名《悼金夫人》）
〔清代〕赵艳雪

逝水韶华去莫留，漫伤林下失风流。
美人自古如名将，不许人间见白头。

> **袁枚《随园诗话·卷四·五一》**
> 冬友侍读出都，过天津查氏，晤佟进士溶；言其母赵夫人苦节能诗，《祭灶》云：
> “再拜东厨司命神，聊将清水饯行尘。年年破屋多灰土，须恕夫亡子幼人。”
> 查恂叔言其叔心谷《悼亡姬》诗，和者甚众。有佟氏姬人名艳雪者，一绝甚佳，其结句云：
> “美人自古如名将，不许人间见白头。”
> 此与宋笠田明府“白发从无到美人”之句相似。
```
